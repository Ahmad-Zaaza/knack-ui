# **Installation**
---

Knack-ui components are written in React and styled-components, and its stories are written in [Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df).

Add Knack-ui to your project.

You can install Knack-UI using NPM or Yarn.

- NPM: `npm install knack-ui --save`
- Yarn : `yarn add knack-ui`
